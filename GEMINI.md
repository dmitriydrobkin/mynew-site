# Мастер-инструкция для ИИ-агентов (Antigravity и др.)

Этот репозиторий — универсальный шаблон Next.js для Cloudflare Pages.
При генерации или изменении кода в этом проекте, **СТРОГО** соблюдайте следующие правила:

1. **Правило 1: Только Edge Runtime.**
   Никогда не использовать Node.js API (fs, path, crypto и т.д.), только Web APIs. Весь проект работает на Cloudflare Edge (v8 isolates). В начале Route Handlers используйте `export const runtime = 'edge';`.

2. **Правило 2: Документация на русском.**
   Любой новый код, функции, интерфейсы TypeScript и API-маршруты должен содержать подробные комментарии на русском языке. Код должен быть предельно понятным.

3. **Правило 3: Масштабируемость и Базы Данных.**
   Архитектура готова к интеграции с Telegram-ботами (см. `/api/telegram-webhook`) и расширению веб-админки. При добавлении базы данных использовать **строго Cloudflare D1** (не PostgreSQL/MySQL напрямую без HTTP-оберток).

4. **Правило 4: SEO в приоритете.**
   Все новые страницы должны иметь корректные семантические HTML-теги и динамически обновлять `metadata` через функцию `generateMetadata`. Всегда поддерживайте `sitemap.ts` и `robots.ts` в актуальном состоянии.
